## Usage Guidelines

### File Formats

- **SVG** - Vector format, ideal for web and scalable applications
- **PNG** - Raster format with low resolution (LR), suitable for quick previews

### Best Practices

- Use SVG files whenever possible for crisp rendering at any size
- Do not alter, distort, or modify the logos
- Maintain appropriate spacing around the logo
- Use the appropriate product logo for its respective context
